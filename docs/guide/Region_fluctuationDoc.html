/*! \page REGION_FLUCTUATIONDoc Calculate the pair distribution function Keyword: REGION_FLUCTUATION \section description Description We define the number operator on as site i for spin \f$ \sigma \f$ as \f$ n_{i,\sigma}= \sum_{j\in \sigma} \Theta(r_j) \f$, where j is summed over the electrons in the spin \f$ \sigma \f$. This object calculates the joint probability distribution function between all pairs of sites and spins, \f$ \rho( n_{i,\sigma},n_{j,\sigma} ) \f$. \section example Example 
method { VMC 
  average { region_fluctuation maxn 5 }
}
